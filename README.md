# NSW Spatial Services Python Examples

Small Python examples showing how surveyors can access NSW Spatial
Services (SIX Maps / Cadastre) using ArcGIS REST APIs.

## Examples

### 1. Lot from Address
Resolve an address to coordinates using Address_Location.

### 2. Nearby Lots
Find lots within a buffer of a given lot/plan.

## Install

pip install -r requirements.txt

## Run

python examples/01_lot_from_address.py
python examples/02_nearby_lots.py
